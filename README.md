# Copy page url

Copu (copy page url) provides to copy page link to clipboard.

![prevew.png](https://raw.githubusercontent.com/ymatzki/copu/master/preview.png)

### Freature

Copu supports following format.

```
# Markdown:
[${title}](${url})

# Confluence wiki:
[${title}|${url}]

# Title and URL:
${title} - ${url}

# Only URL:
${url}
```
